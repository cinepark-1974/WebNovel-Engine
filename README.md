## v2.6 핵심 업데이트 — Profession Pack 통합

### 해결한 문제

**v2.5까지의 약점:**
- 서사 구조·감정·수위·장르 톤은 완벽했으나 **직업 디테일**이 빈약
- 셰프 주인공이 등장해도 "요리를 했다" 수준
- 형사가 등장해도 "수사했다" 수준
- 아이돌이 등장해도 "무대에 올랐다" 수준
- 양질의 웹소설·드라마와의 격차는 바로 이 지점

**v2.6 해결책:**
- Creator Engine에서 검증된 **2,586줄의 Profession Pack** 이식
- 웹소설 특화 4개 카테고리 신규 추가 (재벌·헌터·판타지직업·회빙환)
- 캐릭터 바이블 생성 시 자동 주입
- 회차 집필 시 **등장 캐릭터의 직업 블록만 선택적 주입** (토큰 효율)

## 자동 작동 메커니즘

### 1. 컨셉 카드 단계
```
주인공 직업 입력: "재벌 2세 후계자"
  ↓
[감지된 카테고리]: 재벌2세3세 📚
```

기획서 업로드 시 AI가 자동으로 `profession` 필드 판별해 컨셉 카드에 저장.

### 2. 캐릭터 바이블 단계
```
[버튼 클릭]
  ↓
컨셉 카드에서 주인공·상대역·빌런의 profession 자동 수집
  ↓
각 인물별 Profession Pack 블록 생성 (2,500~5,000자)
  ↓
캐릭터 바이블 프롬프트에 주입
  ↓
[결과] 직업 디테일이 녹아든 캐릭터 프로필
   - 말투에 전문 용어 자연스럽게
   - 행동 패턴에 직업적 습관
   - 공간 디테일에 업무 환경
   - 한국 맥락 반영
```

### 3. 회차 집필 단계
```
EP7 집필 시작
  ↓
해당 회차 등장 인물 추출 (플롯 데이터에서)
  ↓
등장 인물의 직업 블록만 선택적으로 주입 (토큰 효율)
  ↓
집필 프롬프트에 포함
  ↓
[결과] 회차 원고에 자연스럽게 녹아든 직업 디테일
```

### 4. 작가 오류 자동 차단

각 직업 카테고리의 `forbidden` 규칙이 집필 프롬프트에 주입되어, 작가가 흔히 범하는 오류를 차단:

- "검사가 법정에서 직접 피의자 심문" (미국식 오류) → 차단
- "한국 법정에 망치(gavel)" (외국 이미지 오류) → 차단
- "재벌가 안에서 '회장님' 호칭" (호칭 오류) → 차단
- "조직폭력 재계 이름만 바꿔 일반 회사처럼" → 차단
- "마법 체계 없이 남발" → 차단

## 3단계 파이프라인 (v2.6 최신)

```
STEP 1. CONCEPT
   ├── 기획서 업로드 / 아이디어 / 직접 입력
   ├── 모티프 / 페르소나 / 반동인물 / 동일시 전략
   ├── 작품 지향점 (6종 톤 프리셋)
   └── ⭐ 주인공 직업 (23개 카테고리 자동 감지)
          ↓
STEP 2. BUILD-UP
   ├── 2-1. Core Arc 50화 (+ intimacy_schedule)
   ├── 2-2. 떡밥 맵
   ├── 2-3. ⭐ 캐릭터 바이블 (Profession Pack 자동 주입)
   │        - 주인공·상대역·빌런 각자의 직업 블록 통합
   │        - 감지된 카테고리 표시
   ├── 2-4. 회차 플롯 (3~4씬 강제)
   └── 2-5. Extension Arc
          ↓
STEP 3. WRITING
   ├── 3-1. ⭐ 19금 집필
   │        - 작가 페르소나 + 픽션 프레임 + 서사 기능 체크
   │        - 해당 회차 등장 캐릭터의 직업 블록 자동 주입
   │        - forbidden 규칙으로 작가 오류 차단
   ├── 3-2. 15금 변환
   ├── 3-3. 품질 체크
   ├── 3-4. 독자 시뮬레이터
   └── 3-5. 내보내기
```

## 무협 작품 안내

무협 Rule Pack은 **비활성 상태**로 남겨져 있습니다. 무협은 일반 판타지와 완전히 다른:
- 무공 체계 (내공·외공·기경팔맥)
- 문파 구조 (구파일방·마교·흑도)
- 한자 명명법 (초식·심법·신공)
- 기보 시스템
- 강호 세계관

가 필요하므로 **별도 Wuxia Engine** 개발 예정입니다. 아미고 작품·블루진 무협 기획 시점에 개발 착수.
