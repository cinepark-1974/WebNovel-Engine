## v2.3 신기능 — 독자 반영 3층 구조

### 1. 독자 페르소나 시스템 (STEP 1)

타겟 독자를 기획 단계에서 명확히 정의. 5종 프리셋 제공:

| 페르소나 | 소비 동기 | 선호 훅 | 동일시 |
|---------|----------|--------|-------|
| 10대 여성 | 일탈·감수성 | 충격·감수성·아이돌 | 높음 |
| 20~30대 여성 | 스트레스 해소·카타르시스 | 사이다·권력 역전·19금 관능 | 중간 |
| 20~30대 남성 | 성취감·현실 도피 | 먼치킨·레벨업·세계관 | 높음 |
| 40대 이상 여성 | 여운·양질의 서사 | 섬세한 감정선·가족 유대 | 낮음 |
| 청소년 남성 | 오락·세계관 탐험 | 레벨업·던전·시스템 | 매우 높음 |

선택한 페르소나의 소비 동기·선호 훅·동일시 필요도·톤 선호도가 집필 프롬프트에 **자동 주입**.

### 2. 몰입 유도 규칙 (집필 프롬프트 강화)

강종현(2024) 몰입 이론 3단계 조건을 집필 프롬프트에 내장:

**몰입 진입 3대 조건:**
- **명확한 목표 설정** — 3줄 안에 회차 목적 감지 / 도입부에 충격·터부·반전 / 장르 코드 초반 확정
- **도전과 기술의 균형** — 설득력 있는 문장 / 독자-주인공 동일시 장치 / 개연성 강제
- **즉각적 피드백** — 댓글 유발 떡밥 / 참신한 시도 / 결제 의사 유발 훅

**몰입 실패 지점** 3종류(선택 실패 / 진입 실패 / 몰입 실패) 명시적으로 회피.

### 3. 독자 시뮬레이터 (STEP 3 신규 탭)

집필한 회차를 **실제 독자 페르소나 관점**에서 시뮬레이션:

- 🟢/🟡/🔴 몰입 점수 (1~10)
- ✅/❌ 몰입 진입 성공 여부 + 진입 지점
- ✅/❌ 동일시 작동 여부 + 증거
- 💰/🚫 다음 회차 결제 의사 + 이유
- ⚠️ 이탈 가능 지점 지적
- 💬 댓글 유발 포인트

복수 페르소나 동시 시뮬레이션으로 다각 피드백.

### 4. 동일시 전략 필드 (캐릭터 바이블 확장)

주인공 설계 시 독자 몰입 장치를 명시:
- **호칭 처리 방식**: 실명 / 일반명사 통일 / 애칭 중심
- **공감 포인트**: 독자가 공감할 감정·상황 (예: 직장 스트레스, 가족 갈등)
- **내면 독백 스타일**: 냉소적 관찰 / 체념적 사유 등

## 3단계 파이프라인

```
STEP 1. CONCEPT (아이디어 / 기획서 / 직접입력)
   ├── 기획서 업로드 (DOCX/HWP/PDF/TXT 자동 파싱)
   ├── 한 줄 아이디어 → 컨셉 카드 자동 생성
   ├── 9칸 직접 입력 + 서사 모티프 + 🆕 타겟 독자 페르소나 + 🆕 동일시 전략
   ├── 자동 보강 (빌런 / Fatal Flaw / 미스터리 축)
   └── 연재 설정 (플랫폼 · Core/Extension · 수위 · 과금)
          ↓
STEP 2. BUILD-UP
   ├── 2-1. Core Arc (25화 × 2청크, 50화 완결형)
   ├── 2-2. 떡밥 맵
   ├── 2-3. 캐릭터 바이블
   ├── 2-4. 회차 플롯 (🚀 일괄 / 📋 개별)
   └── 2-5. 🆙 Extension Arc
          ↓
STEP 3. WRITING
   ├── 3-1. 19금 원고 집필 (Opus + 페르소나 + 몰입 규칙)
   ├── 3-2. 15금 변환
   ├── 3-3. 품질 체크
   ├── 🆕 3-4. 독자 시뮬레이터 (몰입 피드백)
   └── 3-5. 내보내기 (TXT + DOCX 3종 + ZIP)

💾 전역: 프로젝트 저장/불러오기
```
